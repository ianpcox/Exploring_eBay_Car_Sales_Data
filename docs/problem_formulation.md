# Problem Formulation – Exploring_eBay_Car_Sales_Data

## Research / business question

What features predict price and time-to-sale for used car listings, and how can we provide a baseline for a listing optimizer?

## Primary success metric and threshold

- **RMSE/MAE for price** prediction; time-to-sale as secondary. **Baseline:** Median or mean price by segment (e.g. make).

## Stakeholders and decisions

- **Sellers / dealers:** Decisions about listing attributes and pricing. **Product vision:** Used car listing insights (price/TTS predictions, API for sellers, market trends).
